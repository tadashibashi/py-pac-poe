# Py-Pac-Poe

Console Tic-tac-toe

![screenshot](screenshot.png)

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

## Features

- Player vs. computer mode
- 2 player Mode

## Getting Started

With python 3.10+ installed:
```shell
python main.py
```
